Member Directory HumHub Module
===========================

Member Directory HumHub Module is a sub module of custom_profile module. This module works only when custom_profile module is installed.Member Directory HumHub Module displays the user informations. It uses static profile fields. It cannot be changed.

Each CPP has 3 core pages:
	- Profile view for mentor
	- Profile view for mentee
	- Profile view for each one.

	By selecting other user's profile one can send message to other. And can view linkedin profile of others.



## Installation

- Download Module 
- Put it into protected/modules/
- Make sure custom_profile Module is enabled.
- Enable module under Admin -> Modules
